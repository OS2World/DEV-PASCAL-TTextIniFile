WriteInteger

Writes an Integer value to the file.

Class
TTextIniFile

Declaration 

procedure WriteInteger(const Section, Ident: string; 
                                   Value: LongInt);

Visibility
public

Description
Writes an Integer value to the file. If the destination section does not exist, it is created. If a value with the given identifier already exists, it is overwritten.
