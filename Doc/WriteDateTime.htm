WriteDateTime

Writes a date-and-time value to the file.

Class
TTextIniFile

Declaration 

procedure WriteDateTime(const Section, Ident: string; 
                                    Value: TDateTime);

Visibility
public

Description
Writes a date-and-time value to the file. If the destination section does not exist, it is created. If a value with the given identifier already exists, it is overwritten.
