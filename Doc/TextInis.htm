TextInis

This unit contains a class TTextIniFile that implements textual initialization files.

Description
This unit contains a class TTextIniFile that implements textual initialization files. The files are fully interface compatible with those in the (Delphi) IniFiles unit, but they provide some additional methods for reading and writing data other than String, Integer, and Boolean.
