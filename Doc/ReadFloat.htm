ReadFloat

Reads an floating point value from the file.

Class
TTextIniFile

Declaration 

function ReadFloat(const Section, Ident: string; 
                               const Default: Extended): Extended;

Visibility
public

Description
Reads an floating point value from the file. If either the given section or the given identifier does not exist, or the value cannot be converted to Extended, the default value is returned instead.
