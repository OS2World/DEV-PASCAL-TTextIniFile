WriteChar

Writes a character value to the file.

Class
TTextIniFile

Declaration 

procedure WriteChar(const Section, Ident: string; 
                                Value: Char);

Visibility
public

Description
Writes a character value to the file. If the destination section does not exist, it is created. If a value with the given identifier already exists, it is overwritten.
