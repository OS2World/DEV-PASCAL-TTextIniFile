ReadBool

Reads a Boolean value from the file.

Class
TTextIniFile

Declaration 

function ReadBool(const Section, Ident: string; 
                              Default: Boolean): Boolean;

Visibility
public

Description
Reads a Boolean value from the file. If either the given section or the given identifier does not exist, or the value cannot be converted to Boolean, the default value is returned instead.
