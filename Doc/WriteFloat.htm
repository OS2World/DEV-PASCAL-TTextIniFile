WriteFloat

Writes a floating point value to the file.

Class
TTextIniFile

Declaration 

procedure WriteFloat(const Section, Ident: string; 
                                 Value: Extended);

Visibility
public

Description
Writes a floating point value to the file. If the destination section does not exist, it is created. If a value with the given identifier already exists, it is overwritten.
